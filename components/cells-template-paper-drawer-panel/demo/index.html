Scrolling

<cells-template-paper-drawer-panel/> method scrollItemIntoView() allow to scroll an inner child of main zone.

scrollItemIntoView() payload must be a valid inner child node:

        function fakeScroll(to) {
          var panel = document.getElementById('scroll-panel');
          if (panel) {
            panel.scrollItemIntoView({data: document.getElementById(to)});
          }
        }

Event cells-scroll-complete is fired after completing the scroll, returning the scrolled item.
